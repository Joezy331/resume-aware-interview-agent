# Screenshots — 截图说明

本目录存放项目页面截图，用于 README.md 展示和作品集内嵌。

## 📸 截图清单

| 文件名 | 页面 | 截图内容要求 |
|--------|------|-------------|
| `homepage.png` | 首页 | 完整首屏：标题 + 价值主张 + 使用流程 + 数据信任背书（DAU/满意度） |
| `resume_upload.png` | 简历上传页 | 上传区域 + 解析结果展示（技能标签分类：技术/数据/产品/AI·ML） |
| `interview_page.png` | AI面试页 | 面试进行中截图：左侧问题列表 + 右侧当前问答 + 进度条 |
| `score_report.png` | 评分报告页 | 6维度雷达图 + AI反馈文字 + Human-in-the-loop 反馈按钮 |
| `agent_flow.png`（可选） | Agent流程 | 4个Agent的数据流转示意图（用于README中架构部分） |

## 📐 截图规范

- **尺寸**：推荐 1280×800 px（16:10 比例，MacBook 风格）
- **格式**：PNG（清晰度高，适合展示）
- **工具**：
  - macOS：`Command + Shift + 4`（区域截图）
  - Windows：`Win + Shift + S`（区域截图）
  - 浏览器：Chrome DevTools → `Ctrl+Shift+P` → `Capture full size screenshot`

## 💡 截图建议

1. **截图前清空浏览器地址栏**，避免暴露本地路径
2. **使用 Demo 数据**：简历使用脱敏的示例简历（如 `张三_PM实习生.pdf`）
3. **评分报告截图**：确保雷达图已完整渲染
4. **界面状态**：建议在"刚完成面试"的状态截图，而非空白页
