# Assets — 静态资源说明

本目录存放项目 Logo、图标、Banner 等静态资源。

## 📂 建议存放内容

| 文件名 | 说明 |
|--------|------|
| `logo.png` | 项目 Logo（推荐 200×200 px，透明背景） |
| `banner.png` | GitHub 仓库 Banner（推荐 1280×640 px） |
| `favicon.ico` | 网站图标 |

## 💡 Banner 设计建议

内容元素：
- 项目名称：**Resume-Aware Interview Agent**
- Slogan：*上传简历 → AI解析 → 个性化面试 → 智能评分*
- 技术栈图标：Python / Streamlit / Qwen2.5
- 背景色：深海军蓝（#0a1628）或渐变蓝
